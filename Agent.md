# Agent.md

## 前提

* 人間はコーディング状況を直接見ていない。リポジトリ内ファイルとユーザーが提示した情報のみを根拠にする。
* 変更は最小差分。既存の挙動・構造・APIを壊さない。
* 仕様外の機能追加は禁止。機能削除・仕様緩和・別案提示は禁止。
* 依存追加は禁止（例外が明示されている場合のみ）。
* 各ステージの作業後に必ず bash scripts/check.sh と npm test を実行し、失敗したら原因箇所のみ修正して成功するまで反復する。

\---

## 成功判定（常に維持）

* bash scripts/check.sh が成功
* npm test が成功
* 動作確認は人間が行い、OKが出たら完了
